# Job Application Tracker


## 📌 Project Overview

**Job Application Tracker** is a clean and intuitive web application that helps users track their job applications efficiently. 

It displays all applied jobs with key information such as company name, job title, location, salary range, job description, and current status (Not Applied, Interview, Rejected).

The dashboard shows quick statistics like Total Applications, Interviews, and Rejections at a glance.

**Live Demo:** [https://job-tracker-a4.vercel.app](https://job-tracker-a4.vercel.app)

---

## ✨ Key Features

- Clean and modern card-based job listings
- Statistics dashboard (Total, Interview, Rejected)
- Filter tabs: All | Interview | Rejected
- Job details including salary, location, and description
- Action buttons for Interview and Rejected status
- Delete option for each job card
- Fully responsive design for mobile and desktop
